# Heritage Project
## Overview
The Heritage project is a website built with HTML, CSS, and JavaScript, featuring a coming soon page and a roadmap foundation page.
## Key Features
- Coming soon page with background video and masthead
- Roadmap foundation page with header and footer sections
- Responsive design using Bootstrap
## Tech Stack
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Font Awesome
## Installation
1. Clone the repository: `git clone https://github.com/Heritage/Heritage.git`
2. Open the `index.html` file in a web browser to view the coming soon page
3. Open the `docs/index.html` file in a web browser to view the roadmap foundation page
## Usage
- View the coming soon page to see the background video and masthead
- View the roadmap foundation page to see the header and footer sections

- None required
